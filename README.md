# Análisis de Rendimiento de Tiendas — Proyecto *Alura Store*

Este proyecto analiza el desempeño de cuatro tiendas administradas por el Sr. Juan con el fin de identificar cuál presenta el menor rendimiento global. A partir de esta información, se recomienda cuál tienda debería vender para reinvertir ese capital en un nuevo negocio.

El análisis se realizó en **Google Colab**, utilizando Python, pandas y diversas visualizaciones.

---

##  Propósito del Análisis

El objetivo principal es **evaluar el rendimiento de las cuatro tiendas** mediante métricas clave:

- **Ingresos totales**
- **Categorías más y menos vendidas**
- **Calificaciones promedio de clientes**
- **Productos más y menos vendidos**
- **Coste de envío promedio**

Con base en estos resultados, el proyecto identifica **la tienda con menor desempeño global** para recomendar su venta.
